# RAG Experiments

## 담당
이영준 총괄 / 류종걸 보조

## 목적
Memory Twin의 기억 검색 및 대화 기능을 구현하기 위해 RAG 구조를 실험한다.

## 실험 항목
1. 텍스트 기억 데이터 임베딩
2. Qdrant 저장
3. 질문 기반 유사 기억 검색
4. 검색 결과 기반 LLM 답변 생성
5. 답변 출처 표시
6. 관련 기억이 부족할 때 답변 제한

## MVP 기준
- 사용자의 질문에 대해 관련 기억 3~5개를 검색한다.
- 검색된 기억을 기반으로만 답변한다.
- 답변에는 참조 출처를 표시한다.
- 근거가 부족하면 모른다고 답한다.
