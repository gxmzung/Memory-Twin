# Executive Summary

Memory Twin은 개인의 사진, 음성, 영상, 문서, 대화 기록을 AI가 구조화하여 가족과 다음 세대가 안전하게 보존하고 활용할 수 있도록 돕는 AI 기반 디지털 유산 관리 플랫폼이다.

본 서비스는 사람을 부활시키거나 의식을 재현하는 것이 아니라, 생전 동의 기반의 기록 데이터를 Memory Archive, Memory Graph, RAG 기반 대화 기능으로 연결하여 개인의 기억과 가치관을 보존하는 것을 목표로 한다.

초기 MVP는 웹 기반으로 개발하며, 핵심 기능은 기억 데이터 업로드, 타임라인 생성, 기억 검색, 출처 기반 AI 대화, Legacy Letter, 가족 접근 권한 관리이다.

수익모델은 개인 구독, 가족 플랜, 프리미엄 기능, 장례식장·요양원·보험사 B2B 제휴로 구성한다.
