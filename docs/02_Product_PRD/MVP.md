# Memory Twin MVP

## MVP 정의
Memory Twin MVP는 사용자가 사진, 텍스트, 음성 메모 등 개인 기억 데이터를 업로드하고, AI가 이를 구조화하여 검색 및 대화형 응답으로 제공하는 웹 기반 디지털 유산 관리 서비스이다.

## MVP 핵심 기능
1. 사용자 계정
2. 기억 데이터 업로드
3. Memory Archive
4. Timeline
5. RAG 기반 Memory Chat
6. AI 답변 출처 표시
7. 생전 동의 및 가족 접근 권한 관리
8. Legacy Letter 예약 기능

## MVP 제외 기능
- 고도화된 Voice Clone
- 3D Avatar
- 홀로그램
- 완전한 의식 재현
- 사망자 부활 표현

## 성공 기준
- 사용자가 10개 이상의 기억 데이터를 업로드할 수 있다.
- 업로드된 기억을 시간순으로 볼 수 있다.
- 사용자가 질문하면 관련 기억을 검색해 답변할 수 있다.
- 답변에는 참조된 기억 출처가 표시된다.
- 사용자는 데이터 삭제와 접근 권한을 관리할 수 있다.
