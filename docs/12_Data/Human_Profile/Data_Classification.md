# Data Classification

## 목적

Memory Twin은 모든 데이터를 수집 즉시 등급화한다.  
데이터 등급은 AI 사용 가능 여부, 가족 공유 여부, 다운로드 가능 여부, 삭제 정책을 결정한다.

---

## A등급: 본인 단독 생성 데이터

예시:
- 본인이 쓴 일기
- 본인이 작성한 메모
- 본인이 쓴 편지
- 본인 인터뷰 답변
- 본인 음성 녹취
- 본인 생애 타임라인

AI 사용:
가능

가족 공유:
생전 동의 시 가능

위험도:
낮음

---

## B등급: 본인 중심 데이터

예시:
- 본인 사진
- 본인이 설명한 사진
- 본인 영상
- 본인 발표문
- 본인 블로그 글

AI 사용:
가능하나 초상권·공개 범위 확인 필요

위험도:
중간

---

## C등급: 공동 생성 데이터

예시:
- 카카오톡 대화
- 문자 대화
- 이메일 대화
- 단체 채팅
- 댓글 대화

AI 사용:
제한적 사용

조건:
- 본인 발화만 우선 사용
- 상대방 발화는 기본 제외
- 상대방 동의 시 제한적 사용
- 제3자 정보 마스킹 필수

위험도:
높음

---

## D등급: 제3자 민감정보 포함 데이터

예시:
- 병원 관련 대화
- 가족 분쟁 대화
- 연인과의 사적 대화
- 회사 내부 정보
- 타인의 주소/전화번호
- 타인의 건강 정보

AI 사용:
기본 제외

가족 공유:
기본 차단

위험도:
매우 높음

---

## E등급: 법적 제한 데이터

예시:
- 주민등록증
- 여권
- 금융 비밀번호
- 계약서 원본
- 유언장 원본
- 의료 기록 원본
- 계좌 정보

AI 사용:
금지

저장:
원칙적으로 금지 또는 별도 암호화 보관

위험도:
최고

---

## 핵심 원칙

> Memory Twin은 내가 가진 데이터가 아니라, 내가 사용할 권리가 있는 데이터만 처리한다.
