# AI Memory Cases

## 담당
류종걸

## 조사 목적
Memory Twin의 AI 기억 보존 기능과 유사한 국내외 사례를 조사하여 기능 차별화와 사업계획서 근거로 활용한다.

## 조사 대상
- HereAfter AI
- StoryWorth
- MyHeritage
- Google Photos
- iCloud Photos
- Replika
- Personal AI
- Digital Legacy 서비스
- AI Memory Assistant 서비스

## 비교 항목
| 서비스 | 주요 기능 | AI 대화 | 가족 공유 | 생전 동의 | 유족 권한 | 한계 |
|---|---|---|---|---|---|---|

## 정리 방향
Memory Twin은 단순 사진 저장이나 챗봇이 아니라, 생전 동의 기반으로 기억 데이터를 구조화하고 가족과 다음 세대에 전달하는 디지털 유산 플랫폼이라는 점을 강조한다.
