# 12. Risk Assessment

## 주요 리스크와 대응

| 리스크 | 설명 | 대응 |
|---|---|---|
| AI 환각 | 실제 기록에 없는 답변 생성 | RAG 출처 기반 답변, 근거 부족 시 답변 제한 |
| 사칭 | 실제 인물인 것처럼 오해 | AI 고지 문구 상시 표시 |
| 무단 사용 | 동의 없는 데이터 사용 | 항목별 동의 관리 |
| 가족 분쟁 | 유족 간 권한 충돌 | 제한 모드 및 법적 검토 |
| 심리적 의존 | 유족이 AI에 과도하게 의존 | 안내 문구, 사용 제한, 상담 연계 검토 |
| 데이터 유출 | 민감 기억 데이터 노출 | 암호화, 접근통제, 감사로그 |
| 법적 책임 | AI 답변으로 인한 오해 | 법률·의료·투자 판단 제한 |
| 딥페이크 악용 | 음성/영상 복원 오남용 | 고위험 기능 제한 및 워터마크 |
