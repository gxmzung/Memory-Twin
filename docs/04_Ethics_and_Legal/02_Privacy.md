# 02. Privacy Policy

## 기본 원칙

Memory Twin은 개인의 민감한 기억 데이터를 다루므로 최소 수집, 목적 제한, 암호화 저장, 접근 통제를 기본 원칙으로 한다.

## 개인정보 처리 원칙

1. 필요한 데이터만 수집한다.
2. 수집 목적을 명확히 고지한다.
3. 목적 외 이용을 금지한다.
4. 외부 AI 모델 학습 데이터로 자동 사용하지 않는다.
5. 사용자가 삭제를 요청하면 삭제 절차를 제공한다.
6. 접근 로그를 기록한다.
7. 관리자 접근은 최소화한다.

## 민감 데이터 예시

- 가족 사진
- 음성
- 영상
- 사적인 편지
- 일기
- 의료 관련 기억
- 종교·정치·건강 관련 발언
- 금융 및 상속 관련 문서

## 보호 조치

- 전송 구간 TLS 적용
- 저장 데이터 암호화
- 관리자 접근 로그 기록
- 권한 기반 접근 제어
- 백업 데이터 보관 기간 제한
- 민감 데이터 마스킹
