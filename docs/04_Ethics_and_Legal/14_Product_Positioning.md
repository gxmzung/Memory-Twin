# 14. Product Positioning

## 공식 정의

Memory Twin은 고인을 되살리는 서비스가 아니라, 생전 동의 기반으로 개인의 기억과 디지털 유산을 구조화하여 가족과 다음 세대가 안전하게 보존·탐색·계승할 수 있도록 지원하는 AI 기반 디지털 유산 플랫폼이다.

## 한 줄 설명

사진, 음성, 문서, 가족 기록을 AI가 구조화하여 기억을 보존하고 가족에게 전달하는 디지털 유산 관리 서비스.

## 사업계획서용 표현

Memory Twin은 고령화와 디지털 자산 증가로 인해 발생하는 가족 기억 소실과 사후 데이터 관리 공백을 해결하기 위해, 개인의 기록 데이터를 생전 동의 기반으로 저장·구조화·검색·공유하는 AI 디지털 유산 플랫폼이다.

## 기술계획서용 표현

Memory Twin은 사용자의 기억 데이터를 전처리, 임베딩, 벡터 검색, RAG 기반 생성 과정을 통해 검색 가능한 Memory Archive와 대화형 Memory Chat으로 제공하는 시스템이다.

## 금지 포지셔닝

- AI 부활 서비스
- 고인 복원 서비스
- 영혼 보존 서비스
- 의식 업로드 서비스
- 실제 고인 대화 서비스

## 권장 포지셔닝

- 디지털 유산 관리
- 가족 기억 보존
- 생전 동의 기반 AI 아카이브
- 기록 기반 Memory Chat
- 세대 간 기억 전달 플랫폼
