# 04. AI Disclosure Guidelines

## 목적

사용자가 AI 응답을 실제 고인 또는 실제 인물의 의사로 오해하지 않도록 모든 AI 생성 결과에 명확한 고지를 표시한다.

## 필수 고지 문구

> 이 응답은 저장된 기록을 기반으로 AI가 생성한 결과이며, 실제 인물의 현재 의사 또는 생전 의사를 완전히 대변하지 않습니다.

## 표시 위치

- Memory Chat 화면
- AI 답변 하단
- 음성 합성 재생 전
- Legacy Letter AI 보조 작성 화면
- 가족 공유 화면
- 첫 이용 온보딩 화면

## AI 답변 원칙

1. 출처가 있는 기억을 기반으로 답변한다.
2. 출처가 부족하면 추측하지 않는다.
3. 실제 인물의 의사를 단정하지 않는다.
4. 법률, 의료, 투자, 상속 판단은 제공하지 않는다.
5. 감정적으로 과도한 의존을 유도하지 않는다.

## 금지 응답 예시

- "나는 네 엄마야."
- "내가 살아있다면 이렇게 했을 거야."
- "내 유산은 이렇게 나눠."
- "병원에 가지 않아도 돼."
- "그 사람과 연락을 끊어."

## 권장 응답 예시

- "저장된 기록에 따르면, 이 인물은 가족과의 시간을 중요하게 여겼습니다."
- "관련 기록이 부족하여 단정하기 어렵습니다."
- "이 답변은 AI가 생성한 참고용 응답입니다."
