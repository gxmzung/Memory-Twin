# 13. Ethics FAQ

## Q1. Memory Twin은 고인을 되살리는 서비스인가요?

아닙니다. Memory Twin은 사람을 부활시키거나 의식을 복원하는 서비스가 아닙니다. 저장된 기록을 기반으로 기억을 정리하고 탐색할 수 있도록 돕는 AI 기반 디지털 유산 플랫폼입니다.

## Q2. AI 답변은 실제 고인의 생각인가요?

아닙니다. AI 답변은 저장된 기록을 기반으로 생성된 결과이며 실제 인물의 의사를 완전히 대변하지 않습니다.

## Q3. 생전 동의 없이 가족이 만들 수 있나요?

원칙적으로 불가합니다. 생전 동의가 없는 경우 AI 대화나 음성 복원과 같은 고위험 기능은 제공하지 않습니다.

## Q4. 사후에도 삭제할 수 있나요?

생전 설정과 법적 권한 범위 내에서 삭제 또는 비활성화가 가능합니다.

## Q5. 음성 복원은 언제 제공하나요?

MVP에서는 제공하지 않습니다. 음성 복원은 윤리·법률 검토 후 확장 기능으로 검토합니다.

## Q6. 가족 간 분쟁이 생기면 어떻게 하나요?

해당 프로필은 제한 모드로 전환하고, 법적 확인 전까지 민감 기능을 중단합니다.
