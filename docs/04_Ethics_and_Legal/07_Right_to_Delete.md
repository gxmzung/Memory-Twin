# 07. Right to Delete

## 삭제권 정의

사용자는 본인의 데이터와 AI 활용 여부를 언제든 삭제 또는 비활성화할 권리가 있다.

## 삭제 유형

- 전체 계정 삭제
- 특정 사진 삭제
- 특정 음성 삭제
- 특정 문서 삭제
- AI 대화 기능 비활성화
- 가족 공개 중단
- Legacy Letter 삭제
- 사후 활성화 철회

## 구현 요구사항

- 삭제 API 제공
- 삭제 요청 이력 저장
- 삭제 대상 원본 파일 제거
- Vector DB 내 임베딩 삭제
- 관련 캐시 삭제
- 백업 삭제 스케줄 등록

## 삭제 불가 또는 제한 상황

- 법적 보존 의무가 있는 경우
- 결제 및 정산 기록
- 보안 사고 조사 중인 로그

단, 이 경우에도 서비스 화면에서는 비활성화 처리해야 한다.
