# 05. Family Access Policy

## 목적

가족 구성원이 접근할 수 있는 데이터 범위와 권한을 명확히 정의한다.

## 가족 권한 유형

| 권한 | 설명 |
|---|---|
| Viewer | 공개된 사진, 타임라인 열람 |
| Memory Reader | 텍스트 기록 열람 |
| Voice Listener | 음성 기록 재생 |
| Chat User | AI Memory Chat 이용 |
| Legacy Receiver | 예약 편지 수신 |
| Admin | 가족 권한 관리 |
| Legal Guardian | 법적 대리인 권한 |

## 접근 원칙

- 가족이라도 자동 접근은 허용하지 않는다.
- 생전 지정된 가족 또는 승인된 사용자만 접근 가능하다.
- 접근 권한은 데이터 유형별로 분리한다.
- 모든 접근 기록은 Audit Log에 저장한다.

## 유족 분쟁 대응

- 가족 간 권한 분쟁 발생 시 해당 프로필은 제한 모드로 전환한다.
- 제한 모드에서는 AI 대화, 음성 재생, 데이터 다운로드를 일시 중지한다.
- 법적 확인 전까지 서비스가 임의로 특정 가족에게 권한을 부여하지 않는다.
