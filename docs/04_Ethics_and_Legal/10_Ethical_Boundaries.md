# 10. Ethical Boundaries

## 절대 금지

Memory Twin은 다음 행위를 하지 않는다.

1. 고인 또는 특정 인물이 실제로 살아있다고 표현
2. AI가 실제 의식이나 영혼을 가진다고 주장
3. 생전 동의 없는 음성·영상 복원
4. 유족에게 심리적 의존을 유도
5. 법률·상속·의료·투자 판단 제공
6. 가족 갈등에 대해 단정적 지시 제공
7. 정치·종교·금융 의사결정 대리
8. 타인 사칭
9. 민감정보 무단 공개
10. 출처 없는 기억 생성

## 위험한 기능

- 고인 목소리 실시간 대화
- 영상 아바타
- 감정 유도형 응답
- 장기간 대화 의존 기능
- 자동 조언 기능

## 대응 방식

- 고위험 기능은 v1.0 이후 검토
- 기본 MVP에서는 텍스트 기반 RAG Chat 중심
- 출처 표시와 AI 고지를 기본값으로 적용
- 사용 시간 제한과 휴식 안내 기능 검토
