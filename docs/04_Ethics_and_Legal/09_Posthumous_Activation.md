# 09. Posthumous Activation

## 정의

사후 활성화는 사용자가 생전에 설정한 조건에 따라 사망 이후 Memory Twin 기능 일부를 지정된 가족에게 공개하는 절차이다.

## 활성화 조건

- 생전 사후 활성화 동의 존재
- 지정 가족 또는 법적 대리인의 요청
- 사망 사실 확인 자료 제출
- 내부 검토 완료
- 분쟁 없음 확인

## 활성화 가능 기능

- 공개 타임라인
- Legacy Letter
- 가족 공개 사진
- 제한된 Memory Search
- 제한된 AI Memory Chat

## 활성화 제한 기능

- 생전 동의 없는 AI 대화
- 음성 합성 자동 생성
- 법률·상속 관련 답변
- 민감 문서 공개
- 제3자 개인정보 포함 기록 공개

## 안전장치

- 첫 활성화 전 가족에게 AI 고지 표시
- 모든 응답에 AI 생성 표시
- 가족별 접근 권한 제한
- 유족 분쟁 시 제한 모드 전환
