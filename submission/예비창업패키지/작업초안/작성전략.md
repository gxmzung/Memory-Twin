# 예비창업패키지 작성 전략

## 핵심 포지셔닝
Memory Twin은 AI로 사람을 부활시키는 서비스가 아니라, 생전 동의 기반으로 개인의 디지털 유산과 가족 기억을 구조화하고 보존하는 AI 플랫폼이다.

## 사업계획서 항목 대응
1. 현황
- 대표자 역량
- 팀 구성
- 협약기간 산출물

2. 개요
- Memory Twin 소개
- Problem/Solution/Scale-up/Team 요약

3. Problem
- 고령화
- 가족 기억 소실
- 디지털 유산 관리 공백
- 기존 서비스 한계

4. Solution
- Memory Archive
- Memory Graph
- RAG Chat
- Legacy Letter
- Consent Management

5. Scale-up
- 개인 구독
- 가족 플랜
- B2B 제휴
- 장례/요양/보험 연계
- 글로벌 확장

6. Team
- 개발 총괄
- 사업기획
- UX
- 법률/심리 자문 예정
